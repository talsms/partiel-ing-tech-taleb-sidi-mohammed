Différence entre le Local Storage et le Session Storage : Les deux sont des méthodes pour stocker les donneés dans le navigateur web d'un utilisateur
Le Local Storage : Stocker de manière permanante dans le navigateur de l'utilisateur, même après la fermeture du navigateur ou de l'ordinateur.
Le Session Storage : Stocker les données pour la durée de la session du navigateur de l'utilisateur. Donc de manière temporaire.
